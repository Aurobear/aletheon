# dasein

Phenomenological self-awareness module for the Aletheon agent.

## Overview

Implements the "Dasein" (being-there) concept from existential philosophy, providing the agent with self-awareness and introspection capabilities.

## Architecture

- SelfField — The agent's sense of self
- Consciousness states — Awareness level tracking
- Introspection — Self-observation and reporting

## Key Types

- `SelfField` — Core self-awareness field
- `ConsciousnessState` — Current awareness level
