# RFC-011 核心子系统定义

## Executive（运行核心）

负责： - 生命周期 - 调度 - 监督 - 通信 - 资源 - 权限

拥有状态： - Subsystem State - Resource State - Scheduling State

------------------------------------------------------------------------

## Cognit（认知核心）

负责： - Context Build - Recall Coordination - Reasoning - Planning -
Verification - Reflection - Learning

拥有状态： - Working Context - Deliberation State - Active Plan

------------------------------------------------------------------------

## Dasein（主体核心）

负责： - Identity - Values - Boundary - Care - Narrative - Continuity

建议划分：

Fast Path： - Identity Snapshot - Boundary - Care Weight - Commitments

Deep Path： - Narrative Update - Conflict - Identity Mutation -
Long-term Goal

------------------------------------------------------------------------

## Mnemosyne（经验核心）

负责： - Episodic - Semantic - Procedural - Self Memory - Association -
Consolidation - Replay - Compaction

后台任务： - Memory Decay - Replay - Importance Update

------------------------------------------------------------------------

## Corpus（行动核心）

层次：

Device ↓ Capability ↓ Tool ↓ Workflow ↓ Skill

负责： - Driver - Tool - MCP - Sandbox - Hooks - Skills

------------------------------------------------------------------------

## Metacog（演化核心）

负责： - Genome - Mutation - Candidate - Evaluation - Rollback
